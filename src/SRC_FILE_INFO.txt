Here is a quick breakdown of all the files in this directory of the project.

components:
    Components are the items that are actually rendered into the app-wrapper div in index.html.
    Code is written in JSX.
    File extension is .js or .ts.


style:
    All styles for the website are stored here.
    All are written in SCSS.
    All are imported into main.scss.


bootstrap.js:
    Sets up and maintains the REACTDOM environment in order to render the main component.
    Imports main.scss.
    Pushes compiled content to index.html.
    *Do Not Alter Without Knowledge of What You Are Doing*


vendor.js:
    Ensures correct vendor output.
    *DO NOT ALTER*
